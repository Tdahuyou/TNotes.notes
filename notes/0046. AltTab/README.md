## 1. 📒 概述

- https://alt-tab-macos.netlify.app/
- ![图 0](https://cdn.jsdelivr.net/gh/Tdahuyou/imgs@main/2025-05-24-23-17-18.png)
- macOS 上的一款用于切换应用页面窗口的开源工具。
- AltTab 可以让你向在 Windows 中那样，直接通过 `Alt Tab` 来快速切换不同的应用窗口，并且支持配置。
- 目前 `25.05` 试用下来体验还挺好的。
